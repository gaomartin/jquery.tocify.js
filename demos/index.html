使用指南

该文档是PP云对外开放的，基于标准HTTP RESTFULL协议

视频列表接口

接口定义

接口描述 获取视频列表
接口地址 http://xx.com/v3/api/list
接口协议 HTTP GET
HTTP HEADER Content-Type=application/json

请求参数

名称 说明 数据类型 最大长度 可空
channel_webid 视频webid String 100 是
key 搜索关键字 String 50 是
category_id 分类Id Integer 10 是
page_num 当前页 Integer 5 是
page_size 每页记录数 Integer 5 是
apitk API Token String 40 否

请求示例

http://svc.pptvyun.com/svc/v3/api/channel/channellist?username=ppcloud-api@pptv.com&apitk=9698E2AADD67533696E56A6C3491DA06&pagesize=2

响应内容

名称 数据类型 说明
id id Integer
category_id 分类id Integer
channel_name 视频名称 String
channel_summary 视频描述 String
cover_image 封面图片 String
